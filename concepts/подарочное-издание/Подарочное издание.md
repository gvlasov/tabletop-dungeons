# Упаковка

Покруче, чем базовый набор.

Твёрдая коробка с крышкой, из плотного переплетённого серого картона толщиной 1,5-3 мм

Размер коробки - 19,5см длина x 10см ширина x высота [не продумана, надо решить]

# Содержимое

- Всё, что входит в базовый набор
- Кубик d20
- Фишки для обозначения расы
- Вместо листка с правилами - буклет с правилами, примерная оценка количества страниц - 5. буклет относительно листка дополнительно содержит описание механик рас и более красочные художественные описания мира игры, размер листка 18,5см в высоту x 8,5см в ширину
- Бархатный мешочек 7x9 см для кубика и фишек
